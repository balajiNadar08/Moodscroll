# ✨ Quotes App

> Wisdom, laughter, love,spirituality and inspiration—all in one place.

Discover handpicked quotes across a wide range of categories, from timeless spiritual teachings to unforgettable movie lines. Whether you're seeking motivation, reflection, romance, or a good laugh, there's always a quote waiting for you.
This project is a React Native application built with a focus on performance, clean UI and modular architecture. It includes features like dynamic content rendering, custom styling using Tailwind CSS and optimized list handling.

## 🚀 Features

* 🎲 Explore quotes with a single tap
* ❤️ Save your favorite quotes
* 📱 Clean and intuitive mobile experience
* ⚡ Fast, lightweight, and distraction-free
* 🎨 Clean, dynamic UI
* ♻️ Reusable components
* ⚡ Optimized `FlatList` performance
* 🖼️ Image & video backgrounds
* 📂 JSON-powered content

### Tech Stack

* React Native
* Expo
* TypeScript
* Tailwind CSS (via NativeWind)
* Lucide Icons

### Installation

1. Clone the repository:

   ```bash
   git clone <your-repo-url>
   cd <project-folder>
   ```

2. Install dependencies:

   ```bash
   npm install
   ```

3. Start the development server:

   ```bash
   npx expo start
   ```

### Project Structure

```
/assets        → images, fonts, videos  
/components    → reusable UI components  
/data          → static JSON data (e.g., quotes)  
/screens       → app screens/views  
/styles        → global styles and config  
```
